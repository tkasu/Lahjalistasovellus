Lahjalistasovellus
==================

Lahjalistasovellus on apuwebbisovellus juhlien järjestäjille ja vieraille, 	jolla pystytään koordinoimaan mitä lahjoja järjestäjät toivovat ja mitä lahjaehdotuksia vieraat ovat jo varanneet.
